# Стартовый пакет workout-coach

## Что сделать (2 действия)

1. Распакуй этот архив в папку, например: ~/projects/workout-coach
2. Открой Claude Desktop → вкладка Code → выбери эту папку → вставь промпт ниже

## Промпт для первого сообщения (скопируй целиком)

Прочитай CLAUDE.md. Чанк 1 готов — прогони node test-chunk1.js для проверки
(ожидается 17/17). Сделай git init и коммит "chunk 1: store + exercises".
Затем делай чанк 2 (engine.js) по плану из CLAUDE.md: параметры движка там
зафиксированы. Тесты обязательны, по завершении — коммит и скажи, что нужно
для чанка 3.

## Структура пакета

- CLAUDE.md — контекст проекта (Claude Code читает его каждую сессию)
- docs/ — PRD, архитектура, исходный промпт
- src/js/ — готовый чанк 1: store.js, exercises.js
- test-chunk1.js — тесты чанка 1 (17 шт, все зелёные)
